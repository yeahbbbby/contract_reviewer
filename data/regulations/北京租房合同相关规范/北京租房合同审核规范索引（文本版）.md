# 北京租房合同审核规范索引（优化版）

> **创建目的**：为北京租房合同审核提供系统化的法律规范检索指引  
> **理论框架**：基于《合同起草审查指南：三观四步法》（何力等著，法律出版社2024年第5版）的"宏观-中观-微观"分析框架  
> **更新日期**：2026年4月14日

---

## 索引说明

### 索引维度

本索引采用多维度分类体系：

1. **法律效力层级**：法律 → 行政法规 → 部门规章 → 地方法规 → 地方规章 → 司法解释 → 规范性文件
2. **三观分析法**：宏观-交易结构 / 中观-合同形式 / 微观-合同条款
3. **适用主体**：出租人 / 承租人 / 转租人 / 租赁企业 / 经纪机构
4. **规范主题**：合同效力 / 房屋安全 / 登记备案 / 资金监管 / 短租管理 / 纠纷解决

### 使用说明

建议按照"**三观四步法**"的审查流程使用本索引：
- **第一步**：宏观分析——判断交易可行性（合同类型/主体/标的/程序）
- **第二步**：中观分析——确定合同形式（书面/格式条款/示范文本）
- **第三步**：微观分析——审查具体条款（租金/押金/解除/违约等）
- **第四步**：复核与提交——检查遗漏、沟通确认、提交成果

## 第六部分：合同审核流程建议（基于三观四步法）

### 第一步：宏观分析——交易结构审查

1. **合同类型审查**：确认是否为租赁合同，是否存在其他合同类型成分（如居间、服务）
2. **合同主体审查**：核实出租人是否有权出租（所有权/转租权）、承租人资格、是否存在优先权人
3. **合同标的审查**：房屋是否属于禁止出租范围、是否符合安全标准、是否存在群租风险
4. **合同程序审查**：是否需要登记备案、是否涉及特殊程序（如批准）
5. **交易结构审查**：是否为短租、长租、转租，是否符合区域管理规定

### 第二步：中观分析——合同形式审查

1. **合同形式选择**：书面形式是否符合要求（6个月以上必须书面）
2. **格式条款审查**：是否存在格式条款，是否履行提示说明义务
3. **合同模板使用**：是否参照官方示范文本
4. **预约合同审查**：是否存在预约合同与本约合同的关系
5. **阴阳合同审查**：是否存在规避法律的阴阳合同风险

### 第三步：微观分析——合同条款审查

1. **合同首部**：主体信息、鉴于条款是否准确
2. **房屋信息**：地址、面积、用途、附属设施是否明确
3. **租赁期限**：起止日期、续租、优先承租权是否约定
4. **租金条款**：金额、支付方式、调整机制是否明确
5. **押金条款**：金额、托管、退还条件是否约定
6. **使用维修**：使用限制、维修责任划分是否清晰
7. **转租条款**：是否允许转租、转租条件是否约定
8. **合同解除**：解除条件、提前解除责任是否明确
9. **违约责任**：违约金、赔偿范围是否约定
10. **争议解决**：管辖、法律适用是否约定
11. **其他条款**：优先权、通知送达、不可抗力等

### 第四步：复核与提交

1. **复核**：检查条款之间是否协调、是否存在遗漏
2. **沟通**：与客户确认关键条款、提示重大风险
3. **提交**：提供书面审查意见、保留工作底稿

## 附录：三观四步法理论简介

**"三观四步法"**是《合同起草审查指南》提出的标准化合同起草审查方法：

### 三观

1. **宏观——交易结构**：什么交易？交易能不能做、风险大不大？采取什么交易结构做？
2. **中观——合同形式**：什么文本？采取什么合同形式？
3. **微观——合同条款**：什么条款？合同条款要怎么修订？

### 四步

1. **第一步：沟通需求**——了解交易背景、客户需求
2. **第二步：三观分析**——宏观/中观/微观三个层面全面分析
3. **第三步：复核**——检查遗漏、协调条款
4. **第四步：提交**——提交成果、保留底稿
